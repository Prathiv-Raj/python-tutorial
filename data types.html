DATA TYPES


python data type is divided into two types:

   1.functional data type.
      *numbers.
            1.intger.
            2.float.
            3.complex.
      *boolean.
      *strings.

   2.collection data type.
      *sequence.
            1.list.
            2.tuple.
      *non sequence.
            1.set.
            2.dictinary.

example image:

INT:
     *int only store a whole numbers.
     *the characters are not stored in the int data type.
     Example:
          a=2.


FLOAT:
     *float only store a pointing values.
     *the characters are not stored in the int data type.
     Example:
          a=2.5.


COMPLEX:
     *complex store a imaginary values.
     *the characters are also stored in the int data type.
     Example:
          a=2+3j
