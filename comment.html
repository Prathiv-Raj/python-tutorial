PYTHON COMMENTS

*Comments can be used to explain Python code.
*Comments can be used to make the code more readable.
*Comments can be used to prevent execution when testing code.

type:

  • Single Line Comments

    •    *Single Line comment is used to comment a single.

  • Multi Line Comments

    •    *Multi Line Comment is used to comment a multi lines.

Single Line Comment Example:

#This is a comment
print("Hello, World!")

out put:

Hello, World!

*The # symobl is used to comment the line.
*The line is starting with # this line is not executed.



Multi Line Comment Example:

"""
This is a comment
written in
more than just one line
"""
print("Hello, World!")

out put:

Hello, World!

*The """" symobl is used to comment the multi lines.
*The line is starting and ending is """ this line is not executed.
